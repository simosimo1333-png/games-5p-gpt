# 放課後ダッシュ！

放課後の校舎を舞台に、6人で助け合いながらゴールを目指すスマートフォン向け協力アクションゲームです。

> 現在はコンセプト検証段階です。リポジトリにある `index.html` は、1人用の移動・ジャンプ・ゴール判定を確認する技術プロトタイプであり、6人協力プレイはまだ実装されていません。

## 現在できること

- 左右移動とジャンプ
- 足場との衝突、落下時の復帰
- ゴール判定とクリアタイム表示
- キーボードと画面上のタッチボタンによる操作
- Phaser 3をCDNから読み込む、ビルド不要のブラウザ実行

## すぐに試す

### 現行ゲーム

現行ゲームはビルド不要です。ローカルWebサーバーでリポジトリ直下を配信し、ブラウザで開いてください。

```bash
python -m http.server 8000
```

その後、`http://localhost:8000` を開きます。CDNからPhaserを取得するため、初回実行時はインターネット接続が必要です。

### 新しい開発環境

今後の開発に使用するVite + TypeScriptクライアントは `apps/client` にあります。Node.js 22.12以上を用意し、リポジトリ直下で実行してください。

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開きます。型確認と本番用ビルドは次のコマンドです。

```bash
npm run typecheck
npm run build
```

Pull Requestと `main` への変更では、GitHub Actionsが整形、Lint、型、単体テスト、ビルド、モバイル画面のスモークテストを自動実行します。ローカルでは次のコマンドで主要確認をまとめて実行できます。

```bash
npm run check
```

現在のゲーム機能は、動作を確認しながら新しいクライアントへ段階的に移行します。

### 操作

| 操作 | キーボード | タッチ |
| --- | --- | --- |
| 左右移動 | ← / → | 画面左下の ◀ / ▶ |
| ジャンプ | ↑ / Space | 画面右下の JUMP |
| リトライ | クリア画面をクリック | 「もう一度遊ぶ」をタップ |

## 目指すゲーム体験

- 1セッション3〜5分で遊べる、最大6人のリアルタイム協力プレイ
- プレイヤーごとの役割や能力を組み合わせる攻略
- 全員が置いていかれず、失敗しても助け合って復帰できる設計
- スマートフォンの短時間プレイに適した、直感的な操作と軽快な進行

企画の詳細は [ゲーム企画書](docs/game-design.md) を参照してください。

## 開発方針

1. 既存プロトタイプの挙動を維持したまま、Vite + TypeScriptへ移行する
2. シーン、入力、プレイヤー、ステージ、設定を分離し、テスト可能にする
3. 2人接続の小さなオンライン縦切りを完成させる
4. 権威サーバー方式で6人同期、再接続、ルーム管理を実装する
5. 協力ギミック、演出、計測、運用機能を段階的に追加する

詳細なマイルストーンと完了条件は [開発ロードマップ](docs/roadmap.md) に記載しています。
