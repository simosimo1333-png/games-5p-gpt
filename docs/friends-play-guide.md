# 6人で遊ぶ手順

このゲームは、主催者1人と友人5人の合計6人で遊ぶ想定です。一般公開する必要はありません。

## 主催者が最初に行うこと

1. [公開手順](deployment.md)に従って、ゲーム画面と無料サーバーを公開する。
2. 12文字以上を目安に、6人だけが知る「友人用キー」を決める。
3. ゲーム画面で表示名と友人用キーを入力し、「ルームを作る」を押す。
4. 表示された招待リンクを、5人へ個別のメッセージで送る。

招待リンクには友人用キーが含まれます。SNS、動画、公開掲示板などには載せないでください。

## 友人が行うこと

1. 主催者から届いた招待リンクを、SafariまたはChromeで開く。
2. 自分の表示名を入力する。
3. 「参加する」を押す。
4. 6人全員が揃ったら準備完了にし、主催者がゲームを開始する。

## 無料サーバーの起動待ち

15分ほど誰も使っていないと、無料サーバーは休止します。最初の接続時だけ、起動に最大1分ほどかかる場合があります。ゲーム画面は自動的に接続を繰り返すため、そのまま待ってください。

遊ぶ時刻が決まっている場合は、主催者が5分ほど前にゲーム画面を開き、ルームを作っておくとスムーズです。

## 友人用キーが外部へ漏れた場合

1. Renderの管理画面で `FRIEND_ACCESS_KEY` を新しい値へ変更する。
2. サーバーが再起動したら、新しいキーでルームを作り直す。
3. 新しい招待リンクだけを5人へ送り直す。

古い招待リンクは使えなくなります。
